                                      # AIBPM

                                      AIBPM是一款图形界面的工作流系统。系统采用图形操作界面，使用所见即所得的方式来进行工作流配置，希望打造一个开箱即用的开源产品，帮助非开发人员通过图形操作界面完成的工作流配置与使用、帮助开发人员提供一些思路及代码复用,减少编写代码的时间。

                                      作者按照简洁优雅的原则进行系统设计，简洁方面：作者尽量缩减流程节点种类、数据库表及代码数量。优雅方面：由于作者编码能力一般，只能依靠使用大量作者认为优雅的第三方组件和模块来拉高优雅度。

                                      # 技术

                                      前端采用vue 3.0、Element Plus、Typescript、mxgraph等，后端采用NET 7.0，主要引用开源模块freesql和中台admin框架 ✨

                                      # 功能

                                      系统内置了表单设计器和流程设计器，所有设计器都选用了灵活的框架，方便定制化，流程模板也加入了版本机制，以保证流程的兼容性。
